# 租戶管理

# Tenants List

此頁面顯示租戶下所有租戶空間的基本資訊，可透過點擊 + New Namespace 按鈕新增指定租戶空間。

  • Platform：顯示為雲平台或邊緣平台
  • Namespace：顯示平台中建立的租戶空間
  • Status：顯示為該租戶空間是否為連線/運行狀態
  • IP Address：對外 IP 或 Domain Name
  • Actions：可選擇重新整理 (Refresh) 或刪除 (Delete) 指定租戶空間

# 新增租戶空間

點擊 Tenants頁面中的 + New Namespace 按鈕，即可產生建立租戶空間的視窗。

  • Tenant：選擇租戶
  • Platform：選擇新增租戶空間至雲平台 (Cloud) 或邊緣平台 (Edge: Edge-Name)
  • Namespace：租戶空間名稱
  • IP：租戶空間 IP

註：若需在亞太雲上建立新的租戶空間，請與系統管理員確認並開通IP。

# 租戶空間基本資訊

此頁面顯示租戶空間的基本資訊，點擊 Refresh 按鈕更新基本資訊。

# 應用服務清單

管理雲平台上正在運行的SaaS應用。點擊 Refresh 按鈕更新應用服務清單，點擊 + New Service 新增應用服務，選擇列表項目後再點擊 Actions 可選擇事件 (Events)、容器日誌 (Logs)、容器終端機 (Terminal)、重啟應用服務 (Restart) 或刪除指定應用服務 (Delete)等功能。詳情請參照 SaaS 應用管理 章節的說明。
