# 邊緣端管理

# Edges List

此頁面顯示所有邊緣平台的列表。點擊 + New Edge 按鈕產生建立邊緣平台所需的配置檔，選擇列表項目後再點擊Actions 可選擇重新整理 (Refresh) 或刪除 (Delete) 指定邊緣平台。

  • Edge：顯示邊緣平台名稱
  • Status：顯示為該邊緣平台是否為連線狀態

# 新增邊緣平台

點擊Edges頁面中的 + New Edge 按鈕，即可產生建立邊緣平台的視窗。

點擊 OK 確認下載後，即可獲得部署邊緣平台所需的配置檔 edge-name.cfg。取得配置檔後，即可透過 Ansible 進行遠端部署邊緣平台。

# 邊緣平台基本資訊

此頁面顯示邊緣平台的基本資訊，點擊 Refresh 按鈕更新基本資訊。

# 節點清單

此頁面顯示邊緣平台上所有的 k8s node，點擊 Refresh 按鈕更新節點清單。

  • Name：節點名稱
  • Alias：節點別名
  • IP：節點 IP

# 租戶空間清單

# 清單頁面介紹

此頁面顯示目前邊緣平台中已經建立的租戶空間。點擊 Refresh 按鈕更新租戶空間清單，點擊 + New Namespace 新增 租戶空間，選擇列表項目後再點擊 Actions 可選擇匯入憑證 (Certificate) 或刪除 (Delete) 租戶空間。

  • Tenant：租戶名稱
  • Name：租戶空間名稱
  • Alias：租戶空間別名
  • IP：租戶空間 IP

# 新增租戶空間

邊緣平台中的租戶空間可在以下兩個頁面中新增：

  • Tenants頁面：點擊 Tenants頁面中的 + New Namespace 按鈕，即可產生建立租戶空間的視窗。

    • Tenant：選擇租戶
    • Platform：選擇新增租戶空間至雲平台 (Cloud) 或邊緣平台。
    • Namespace：租戶空間名稱
    • IP：租戶空間 IP

  • Edges頁面：點擊 Edges頁面中的 + New Namespace 按鈕，即可產生建立租戶空間的視窗。

    • Tenant：選擇租戶
    • Namespace：租戶空間名稱
    • IP：租戶空間 IP

# 應用服務清單

管理邊緣平台上正在運行的SaaS應用。點擊 Refresh 按鈕更新應用服務清單，點擊 + New Service 新增應用服務，選擇列表項目後再點擊 Actions 可選擇事件 (Events)、容器日誌 (Logs)、容器終端機 (Terminal)、重啟應用服務 (Restart) 或刪除指定應用服務 (Delete)等功能。詳情請參照 SaaS 應用管理 章節的說明。
