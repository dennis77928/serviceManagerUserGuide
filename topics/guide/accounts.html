# 帳戶管理

# Account List

此頁面顯示帳號列表，只有具有管理者(Admin)權限的角色可以操作此頁面。點擊 + New Account 按鈕產生登入所需的Token，選擇列表項目後再點擊 Actions 可選擇編輯 (Edit) 或刪除 (Delete) 指定 Token。

  • Token：顯示已創建的 Token 清單
  • Role：顯示 Token 的角色權限，可分為 admin 或 developer
  • Comment：顯示與 Token 對應的註解

# 新增帳號

在Accounts頁面中，點擊 + New Account 按鈕後，便會產生視窗。在視窗中，可以編輯登入所需的 Token 的部署權限。預設為UI操作模式，可以透過右上角開關切換為進階編輯模式。編輯完成，點擊 Submit 即送出。

  • role：可選擇 admin 或 developer。其中 admin 具有管理角色、建立邊緣平台與建立租戶空間的能力。
  • allow_tenant：允許操作的租戶ID，使用星號(*)表示允許操作全部租戶。
  • allow_app：允許操作的應用服務名稱，使用星號(*)表示允許部屬全部應用服務。
  • allow_ns：允許操作的租戶空間。若租戶空間在雲端平台上，則使用 cloud:namespace 表示；若租戶空間在邊緣平台上，則使用 edge:edge_name:namespace 表示。

範例

  • 若雲端平台有一個租戶空間 (flowring)，則寫成 cloud:flowring。
  • 若邊緣平台 (hc-edge) 有一個租戶空間 (flowring)，則寫成 edge:hc-edge:flowring。

註：allow_ns 亦可使用星號(*)表示可使用該平台中的所有租戶空間。

  • comment：該帳號的註解提示。
  • allow_sys：允許操作的系統，請選擇 smui。

# 編輯帳號

點擊指定 Token 的 Actions 中的 Edit 後，會出現編輯視窗。預設為UI操作模式，可以透過右上角開關切換為進階編輯模式。編輯完成後，點擊 Submit 即送出。

# 刪除帳號

點擊指定 Token 的 Actions 中的 Delete 後，會出現彈窗詢問是否刪除 Token。點擊 OK 即開始刪除。
