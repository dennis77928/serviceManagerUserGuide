# Service Manager 介紹

# 目的

Service Manager (opens new window) 主要目的為提供一整合性工具，可同時控管 CorePro 平台上雲端和所有邊緣端節點的整合性管理系統。Service Manager 由專業的維運人員操作，依照客戶需求，部署所需的應用程式至雲平台或霧平台。例如一個典型的IOT系統的建置通常會包含多方的合作，包括應用開發商、設備商、平台服務等。

# 主要功能

Service Manager 是整個雲平台的中控系統，提供運維人員管理、佈署、刪除與更新應用服務至雲端平台和邊緣平台上的工具。運維人員可以直接在 UI 上點擊相關應用服務，即可進行上述操作。

  • 維運人員可以執行：

    • 雲 / 邊緣端節點控管
    • SaaS 應用安裝 / 移除 / 維護

  • CorePro 平台最高管理者，除了上述三點外，還可以進行：

    • 帳戶權限管理
    • 儲存空間管理
    • 軟體授權管理
    • 系統設定

# 帳號申請

Service Manager 管理者的 token，應於安裝時就已綁定，無須額外申請。但若想開放給其他人使用，則需要以管理者 token 登入，並於 Accounts 的分頁內建立 token 給其他人使用。
