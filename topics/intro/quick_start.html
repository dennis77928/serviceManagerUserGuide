# 快速入門

本章節將協助您部署一個應用服務，詳細的操作與設定請參照 SaaS 應用管理 章節的說明。

前提條件

我們假設您已擁有管理者 (Admin) 或開發者 (Developer) 的 token，並且已經登入 Service Manager 平台。

  1. 進入應用服務清單頁面，並點擊 + New Service 新增應用服務

  2. 在 App 清單中，找到您要部署的應用服務名稱，並點擊 Deploy 按鈕。

  3. 出現安裝資訊的彈窗後，在 Gereral 分頁中，填寫 Storage Name，並且按下 OK 開始部署。

  4. 等待安裝進度條跑完，則表示此次部署已完成。
